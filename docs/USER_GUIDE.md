# User Guide

## Introduction
This guide provides instructions for setting up and using the Analysis tool.

## Setup Instructions

1. **Prerequisites**
   - Ensure that you have the following software installed:
     - Python 3.x
     - Git

2. **Clone the Repository**
   ```bash
   git clone https://github.com/jakesteinmetz1722/Analysis.git
   cd Analysis
   ```

3. **Install Dependencies**
   - Use pip to install the required packages.
   ```bash
   pip install -r requirements.txt
   ```

## Usage Guidelines

1. **Running the Analysis Tool**
   - To run the tool, use the following command:
   ```bash
   python main.py
   ```

2. **Configuration**
   - Modify the configuration file located in `config/config.yaml` to set your preferences.
